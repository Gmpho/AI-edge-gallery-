# Strike Tips — Setup Guide

## What You Need
- An Android phone (Android 12+)
- ~2-8 GB free storage (for AI models)
- Wi-Fi (models are large files)

## Step 1: Install the APK

1. Download `striketips.apk` from the landing page
2. Open the file when it finishes downloading
3. If prompted, enable **"Install from unknown sources"**
   - Settings → Security → Install unknown apps
   - Select your browser → toggle "Allow from this source"
4. Tap **Install**

## Step 2: Download an AI Model

1. Open **Strike Tips**
2. Tap the **Model Manager** card on the home screen
3. Browse or search for a model (recommended: **Gemma 2B** for a good balance of speed and capability)
4. Tap **Download** (this happens entirely on your device — private and secure)
5. Wait for the download to complete

## Step 3: Use Racing Skills

1. From the home screen, tap **Agent Skills**
2. Tap the **+** icon
3. Select **Add Skill from URL**
4. Paste a skill URL from the list below
5. Tap **Save**
6. Go back to Agent Chat and start using the skill

## Step 4: Connect the MCP Server (Optional)

For advanced racing intelligence (AI search, odds, race verification):

1. From the home screen, tap **Agent Skills**
2. Tap **MCP Servers**
3. Tap **Add Server from URL**
4. Paste the MCP URL: `https://striketips-mcp.gmphorg379.workers.dev/mcp`
5. Tap **Save**
6. The server tools will appear in Agent Chat after you select it

## Step 5: Start Using It

- Ask the AI to calculate a probability edge
- Ask it to suggest a stake size
- Ask it to analyze a race
- Use the Paper Trader to track virtual bets

## Privacy

- All AI processing happens 100% on your device
- No data is sent to any server
- The MCP server does not store or log your queries
- No account, no registration, no tracking

## Responsible Gambling

Horse racing and betting involve financial risk. Strike Tips is a tool for analysis and education — not a guarantee of winning. Never bet more than you can afford to lose.
